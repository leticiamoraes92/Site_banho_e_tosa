# 🐾 PetCare - Banho e Tosa

Site institucional desenvolvido para um serviço de banho e tosa para pets, com o objetivo de apresentar os serviços oferecidos, permitir agendamento online e disponibilizar informações de contato.

Este projeto foi criado como prática de desenvolvimento web durante os estudos de **Análise e Desenvolvimento de Sistemas**.

---

## 🚀 Tecnologias Utilizadas

- 🌐 HTML5 – Estrutura das páginas  
- 🎨 CSS3 – Estilização e layout  
- ⚡ JavaScript – Interações básicas  
- 🗂 Git – Controle de versão  
- 💻 GitHub – Hospedagem do repositório
